# McDonald-s-Menu-Clustering
Cluster and Principal Component Analysis of McDonald's menu

This is a Clustering and PCA analysis of McDonald's nutritional menu, splitting the data into McDonald's drinks and food before analyzing clusters for each.

The report is found here: https://github.com/thomaskwok91/McDonald-s-Menu-Clustering/blob/master/Stat717_Thomas_Kwok_exam.pdf

The code for the paper is found here: https://github.com/thomaskwok91/McDonald-s-Menu-Clustering/blob/master/McDonalds_Code-for-paper.R

Additional exploration code is found here: https://github.com/thomaskwok91/McDonald-s-Menu-Clustering/blob/master/McDonalds_Exploration-Codes.R
